## 🚀 **为什么加入？**  
- 用最快乐的方式练习编程 **快乐是编程的根本动力**
- 无需严肃的代码审查，**好玩是第一生产力**  

## 📦 **快速开始**  
```bash
# 克隆仓库（需要 Rust ≥1.80）
git clone https://github.com/boren07/fun-cli.git
cd fun-cli

# 编译并安装
cargo build --release
cargo install --path .  # 安装到全局

# 运行单元测试
cargo test -- --nocapture  # 禁止输出被吞

# 恭喜你！可以开始愉快的玩耍了
fun -h    # 查看帮助


```
